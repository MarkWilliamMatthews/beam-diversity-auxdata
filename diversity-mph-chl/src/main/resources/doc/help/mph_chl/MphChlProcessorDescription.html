Diversity Mph/Chl Processor - Processor Description

Processor user interface

The MPH processor is invoked from the VISAT tool menu by selecting the MPH Processor command. On the command line the MPH processor is available by means of the Graph Processing Tool gpt which is located in the BEAM bin directory. Typing gpt Diversity.MPH.CHL -h displays further information.

Selecting the Diversity.MPH.CHL Processor command from the VISAT tool menu pops up the following dialog:

Source Product Group

Source Product: Here the user specifies the source product consisting of bottom-of-Rayleigh corrected reflectances. The combo box presents a list of all products open in VISAT. The user may select one of these or, by clicking on the button next to the combo box, choose a product from the file system.

Target Product Group:

Name: Used to specify the name of the target product

Save as: Used to specify whether the target product should be saved to the file system. The combo box presents a list of available file formats. The text field or the button next to it allow to specify a target directory.

Open in VISAT: Used to specify whether the target product should be opened in VISAT. When the target product is not saved, it is opened in VISAT automatically.

Processing Parameters Panel

Valid pixel expression: An expression which identifies valid input pixels according to flags that are available in the input image.

Cyano max value: Threshold to cap maximum chlorophyll concentration estimated for cyanobacteria-dominant waters.

Chl thresh for float flag: Value, above which cyanobacteria scum is expected, even if not indicated by a BRR maximum at 753 nm (see Algorithm specification, Figure 2).

Export mph: Write the internally calculated MPH value to a separate band for the evaluation of custom peak height to concentration conversions.
